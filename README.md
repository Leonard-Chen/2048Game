# 基于C# WinForms实现的2048小游戏

## 项目简介

这是一个使用 C# 和 WinForms 框架开发的 2048 小游戏。游戏的玩法是通过WASD（或者↑↓←→）键操作整个棋盘的所有方块往某个方向滑动，两个相同数字的方块在碰撞时数字会相加。每次滑动后，系统会随机选取一个空白格子生成数字2。当棋盘没有空白格子，并且找不到相邻的两个相同数字的方块时，游戏结束。游戏的目标是不断地让相同数字的方块合并，直到合成出2048。

除了基本的游戏功能外，还提供了以下两种功能：

1. **游戏设置**：用户可以自定义棋盘的行数和列数（介于 2~10 之间）。
2. **游戏读档/存档**：用户可以在中途关闭游戏时选择是否保存游戏进度，并在下次启动游戏时选择继续游戏或开始新游戏。

## 功能特性

- **自定义棋盘大小**：用户可以根据自己的喜好设置棋盘的行数和列数。
- **游戏存档**：支持保存和加载游戏进度，方便用户随时继续游戏。
- **简洁的界面**：使用 WinForms 框架开发，界面简洁易用。

## 如何运行

1. **克隆项目**：首先，将项目克隆到本地。
   ```bash
   git clone https://github.com/Leonard-Chen/2048Game
   ```

2. **打开项目**：使用 Visual Studio 2022 打开解决方案 2048.sln。

3. **编译运行**：按下 Ctrl+F5 编译并运行项目。

## 使用说明

### 主界面

- **新游戏**：点击“开始游戏”按钮，可以选择是否覆盖原有存档并开始新的游戏。
- **继续游戏**：点击“继续游戏”按钮，可以加载上次保存的游戏进度。

### 游戏设置

- 在游戏设置中，用户可以自定义棋盘的行数和列数，范围在 2~10 之间。
- 设置界面有显示历史游戏最高分，用户可以选择清空。

### 游戏存档

- 在游戏过程中，用户可以随时关闭游戏窗口，系统会提示是否保存当前游戏进度。
- 下次启动游戏时，用户可以选择继续上次保存的游戏或开始新游戏。
